# ModelAI

ModelAI is an open-source platform dedicated to hosting artificially intelligent generated voices.

## How to Use

### Install dependencies

```bash
npm install
```

### Generate types

```bash
npm run gen:types
```

### Run the development server

```bash
npm run dev
```
